How does it works?

1

STFT Feature Extraction

with Librosa STFT

음원 데이터 분석에 적합한
FFT의 하위 분류인
STFT를 활용해 STFT 특징 추출

2

Beat State Extraction

with Keras B-LSTM

대용량 시계열 데이터 분석에 적합한
RNN의 하위 분류인
LSTM을 활용해 박 상태 추출

3

Beat Extraction

with Procedual Error Compensation

절차적 오차 보정을 통해
박 상태 유지 시간에 따른 박 판별
